# signal tools


## Summary 
"signal tools" aims to provide a Docker container with the most importand lab tools for working with signals of any kind, including recording and processing.

## Software
* rf
	* gnuradio
		* gr-baz
	* gqrx
* Networking
	* Wireshark
* misc
	* vim
	* scipy
